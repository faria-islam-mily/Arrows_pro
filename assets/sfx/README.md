# Sound effects

Drop short `.mp3` clips here with these exact names and the game will play them
(gated by the **Sound** toggle in Settings):

| File          | When it plays            | Suggested feel                |
|---------------|--------------------------|-------------------------------|
| `pop.mp3`     | an arrow slides off      | tiny whoosh / tick (~150 ms)  |
| `blocked.mp3` | a blocked tap (−1 life)  | muted low click               |
| `win.mp3`     | level complete           | soft 3–4 note ascending chime |

Keep them small and quiet; the game mixes them at reduced volume.
Until you add files, sound is simply a no-op (no errors).
